点击【复用】按钮、【智能扫描】按钮、【编号完成】按钮后要向后端发送请求

## 点击【复用】按钮

请求后端id为 scanDeviceState 的接口

- request params

  ```json
  {
    "algorithmProcessId": "scanDeviceState",
    "param": {
      "data": {
        "type": "reset"
      }
    }
  }
  ```

- response params

## 点击【编号完成】按钮

  ```json

    "algorithmProcessId": "saveDeviceArrange",
    "param": {
      "data": {
        "deviceArrange": [

            "columnNumber": "1",
            "id": "1",
  ```
