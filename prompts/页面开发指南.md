# 页面开发指南

基于屏保页面的实现，本文档说明如何创建一个新页面。

## 完整开发步骤

### 1. API 层配置

#### 1.1 配置算法过程 ID - `src/api/client/config.js`

在 `ALGORITHM_PROCESS_IDS` 中添加新接口的 ID：

```javascript
export const ALGORITHM_PROCESS_IDS = {
  HOME_OVERVIEW: 'queryHomePageData',
  SCREEN_DATA: 'queryScreenData',
  XXX_DATA: 'queryXxxData',  // 新增
}
```

#### 1.2 创建数据适配器 - `src/api/adapters/xxx.js`

```javascript
function toText(value, fallback) {
  if (value == null || value === '') {
    return fallback
  }
  return String(value)
}

export function createDefaultXxxData() {
  return {
    field1: '默认值1',
    field2: '默认值2',
  }
}

export function adaptXxxData(rawData) {
  const fallback = createDefaultXxxData()
  // 处理统一返回结构 { code, data, msg, success }
  const responseData = rawData?.data ?? rawData
  const source = responseData?.data ?? responseData ?? {}

  return {
    field1: toText(source.backendField1, fallback.field1),
    field2: toText(source.backendField2, fallback.field2),
  }
}
```

#### 1.3 添加 API 调用函数 - `src/api/modules/home.js`

```javascript
import { ALGORITHM_PROCESS_IDS } from '../client/config'
import { adaptXxxData } from '../adapters/xxx'

export async function queryXxxData(paramData = {}) {
  const response = await callAlgorithmProcess(ALGORITHM_PROCESS_IDS.XXX_DATA, paramData)
  return {
    ...response,
    data: adaptXxxData(response.data)
  }
}
```

### 2. 数据层 - 创建 Hook

#### 创建 `src/features/xxx/hooks/useXxxDataQuery.js`

```javascript
import { createDataQuery } from '@/shared/hooks/createDataQuery'
import { adaptXxxData, createDefaultXxxData } from '@/api/adapters/xxx'
import { queryXxxData } from '@/api/modules/home'

export const useXxxDataQuery = createDataQuery({
  queryKey: ['xxx-data'],
  queryFn: queryXxxData,
  select: (response) => adaptXxxData(response?.data ?? response),
  createDefaultData: createDefaultXxxData,
})
```

### 3. 页面组件

#### 创建 `src/pages/XxxPage.jsx`

```javascript
import { useLocation } from 'react-router-dom'
import './XxxPage.css'
import { useXxxDataQuery } from '@/features/xxx/hooks/useXxxDataQuery'

function XxxPage() {
  const location = useLocation()
  const isXxxRoute = location.pathname === '/xxx'
  const { data: xxxData } = useXxxDataQuery({ enabled: isXxxRoute })

  return (
    <div className="xxx-page">
      {/* 页面内容 */}
    </div>
  )
}

export default XxxPage
```

#### 创建 `src/pages/XxxPage.css`

```css
.xxx-page {
  width: 100%;
  height: 100%;
  min-height: 0;
}
```

### 4. 路由配置

在 `src/App.jsx` 中添加路由：

```javascript
import XxxPage from './pages/XxxPage'

// 在 Routes 中添加
<Route path="/xxx" element={<XxxPage />} />
```

## 关键实现模式

### 1. 统一接口格式

所有后端请求使用统一 POST 接口：

**接口路径：** `/FinforWorx/algorithm/process/execute`

**请求格式：**
```json
{
  "algorithmProcessId": "queryXxxData",
  "param": {
    "data": {}
  }
}
```

**响应格式：**
```json
{
  "code": 200,
  "msg": "执行成功!",
  "data": {},
  "success": true
}
```

### 2. Token 机制

- Token 存储在 `sessionStorage` 中，key 为 `'token'`
- 请求时自动注入到请求头 `token` 字段
- 开发阶段可在 `auth.js` 中配置 `TEST_TOKEN` 进行测试

### 3. 轮询控制 - 页面级启停

每个页面的轮询只在该页面激活时运行：

```javascript
const location = useLocation()
const isXxxRoute = location.pathname === '/xxx'
const { data } = useXxxDataQuery({ enabled: isXxxRoute })
```

### 4. 数据适配 - 字段映射 + 兜底

适配器负责：
- 后端字段名 → 前端字段名映射
- 类型统一（转字符串/数字）
- 提供默认值，防止数据缺失

### 5. 路径别名 - 使用 `@/`

所有导入使用 `@/` 前缀，避免 `../../..`：

```javascript
import { xxx } from '@/api/modules/home'
```

### 6. 工厂函数 - `createDataQuery`

统一处理：
- 保存最后成功数据
- React Query 配置
- 降级逻辑

## 注意事项

1. **轮询间隔**：默认 10 秒，可在 `createDataQuery` 中调整
2. **错误处理**：失败时自动显示上次成功数据
3. **路径别名**：已配置 `@` 指向 `src` 目录，使用 `@/` 替代 `../../..`
4. **组件复用**：优先使用现有组件（如 ScreenInfo、HomeWidget 等）
5. **CORS 预检**：跨域请求会先发 OPTIONS 预检请求，这是正常浏览器行为，不影响性能
