# 向导模块剩余页面开发

根据Figma原型图，生成系统选择与配置页面、项目基本信息页面、区域选择页面、热泵侧循环水泵选择与配置页面和末端侧循环水泵选择与配置页面。

## 系统选择与配置页面

figma链接：https://www.figma.com/design/tYdPFcntfF6bHPhyyYTelM/CAS%E4%BA%A4%E4%BB%98%E6%96%87%E6%A1%A3?node-id=90-46994&m=dev

注意：

- 在空气源热泵台数模块中的滑块，在项目代码中有可复用的组件，直接用即可；（空气源热泵台数模块最大50台）

- 项目需求类型模块、供暖末端类型模块、系统循环形式模块、耦合能源类型模块和耦合能源设备台数的按钮做成组件；组件分为单选和多选；组件数据做成可配置对象：可以增加或删除数据；

- 返回按钮去掉，只留下一步按钮，点击下一步，发送请求至id为saveSystemConfig的后端接口中：

  - 请求参数为：

    ```json
    {
      "algorithmProcessId": "saveSystemConfig",
      "param": {
        "data": {
          "projectData": {
            "projectTypeId": "1",
            "heatPump": "7",
            "terminalTypeId": "1",
            "coupleEnergyTypeId": "2",
            "systemTypeId": "1",
            "coupleEnergyNumber": "1"
          }
        }
      }
    }
    ```

  - 返回参数为：

    ```json
    {
      "code": 200,
      "msg": "执行成功!",
      "data": {
        "state": "success",
        "message": "保存成功"
      },
      "success": true
    }
    ```

    如果返回参数中 data.state值为success，代表保存成功，将systemTypeId存进状态变量中，进入项目基本信息页面；否则保存失败，进不了下一个页面

## 热泵侧循环水泵选择与配置页面

figma链接：https://www.figma.com/design/tYdPFcntfF6bHPhyyYTelM/CAS%E4%BA%A4%E4%BB%98%E6%96%87%E6%A1%A3?node-id=90-48107&m=dev

注意：

- 主循环泵总数模块和备用循环泵台数模块中的滑块，在项目代码中有可复用的组件，直接用即可；（主循环泵总数和备用循环泵台数最大为5台）

- 循环泵运行模式模块的组件，用系统选择与配置页面的按钮组件即可；

- 该页面用到的图片在 assets/guide-init/ 内

- 点击返回按钮，回退到区域选择页面；

- 点击下一步按钮，发送请求至后端id为saveBeginCirculatingPumpConfig的接口：

  - 请求参数为：

    ```json
    {
      "algorithmProcessId": "saveBeginCirculatingPumpConfig",
      "param": {
        "data": {
          "projectData": {
            "heatCirculationPumpMain": "2",
            "heatCirculationPumpSpare": "1",
            "heatCirculationPumpMode": "定频"
          }
        }
      }
    }
    ```

  - 返回参数为：

    ```json
    {
      "code": 200,
      "msg": "执行成功!",
      "data": {
        "state": "success",
        "message": "保存成功"
      },
      "success": true
    }
    ```

    如果返回参数中 data.state值为success，代表保存成功，如果systemTypeId为2，则进入末端循环泵页面，否则进入热泵布局页面（该页面先占位）；否则保存失败，进不了任何一个页面
